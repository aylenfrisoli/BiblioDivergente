Manifiesto · 2026

BiblioDivergente nació de una pregunta simple: ¿por qué los textos no se adaptan a las personas?

Diseñado con Diseño Universal · Construido con empatía

Para quienes tienen TDAH, dislexia, TEA o simplemente leen diferente, la web estándar es una barrera. Esta herramienta es la respuesta.

Cada mente procesa la información de manera única. Algunas necesitan menos ruido visual. Otras, oraciones más cortas. Otras, simplemente más tiempo.

Ninguna forma de leer es incorrecta: el problema siempre fue que los textos no se adaptaban a las personas. Hoy, eso cambia.
